
# Mode303 Luxury Node Website

## Run locally
```bash
npm install
npm start
```

Open http://localhost:3000

## Deploy to Fly.io
- Push this folder to GitHub
- Create app in Fly.io or use `flyctl launch`
- Deploy

## Files
- `index.js` : Express server
- `public/index.html` : Main page
- `public/style.css` : Styling
